Canvas HTML5

Seminario UPS “Desarrollo Web con HTML5, CSS3 y JavaScript”



Integrantes: Iván Marriott - Kevin Marin - Félix Gavilanes - Mario Pincay
Canvas es un nuevo elemento de HTML5 que nos permite dibujar gráficos con Javascript.
Nosotros podemos añadir un canvas a nuestra página de esta forma:

< canvas id="canvas" width="800" height="200" >< canvas >
Ejemplo

Como vemos en este ejemplo la panorámica se mueve hacia
el lado derecho. La imagen tiene la medida de 1024x198 y el canvas 800x200.

La imagen debe ser mucho más grande que el canvas para de este modo realizar el movimiento

Condicionando el tamaño la imagen se va moviendo sobre el lienzo llamado CANVAS.

El potencial de canvas, reside en su habilidadpara actualizar su contenido entiempo real.

Canvas fue creado por Apple y luego fue estandarizado para incorporarlo a HTML5.

Las aplicaciones de Canvas pueden ser grandes como Juegos, efectos dinámicos, aplicaciones.

Y son muchas mas aplicaciones que podremos encontrar en Github con esta gran etiqueta CANVAS
